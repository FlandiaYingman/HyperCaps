# HyperCaps

HyperCaps is an AutoHotKey v2 script that enhances the CapsLock key on Windows.

I created this because I don't want to move my fingers on the keyboard to reach the arrow keys, home, end, and other keys. I wanted to have a quick way to navigate through text and code without moving my fingers outside the main keyboard area. 

| Shortcut                           | Output                                               |
| ---------------------------------- | ---------------------------------------------------- |
| <kbd>CapsLock</kbd> + <kbd>I</kbd> | Up (<kbd>↑</kbd>)                                    |
| <kbd>CapsLock</kbd> + <kbd>J</kbd> | Left (<kbd>←</kbd>)                                  |
| <kbd>CapsLock</kbd> + <kbd>K</kbd> | Down (<kbd>↓</kbd>)                                  |
| <kbd>CapsLock</kbd> + <kbd>L</kbd> | Right (<kbd>→</kbd>)                                 |
| <kbd>CapsLock</kbd> + <kbd>U</kbd> | Jump to Line Begin (<kbd>Home</kbd>)                 |
| <kbd>CapsLock</kbd> + <kbd>O</kbd> | Jump to Line End (<kbd>End</kbd>)                    |
| <kbd>CapsLock</kbd> + <kbd>Y</kbd> | Jump to Previous Word (<kbd>Ctrl</kbd>+<kbd>←</kbd>) |
| <kbd>CapsLock</kbd> + <kbd>;</kbd> | Jump to Next Word (<kbd>Ctrl</kbd>+<kbd>→</kbd>)     |

All the shortcuts can be used with other modifiers. For example, they can be used with <kbd>Shift</kbd> to select text.

Moreover, it is suggested to configure the keyboard repeat delay to the minimum and the repeat rate to the maximum. This can be done in the Windows settings of "Keyboard". This will make your navigation even faster.
